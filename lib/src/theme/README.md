# lib/src/theme

Sonus Auris branding for the Flutter app, mirrored from the marketing site so
the app and web feel like one product.

- **[sonus_theme.dart](sonus_theme.dart)** — brand colour palette (green +
  orange) and the assembled Material `ThemeData`.
- **[sonus_brand.dart](sonus_brand.dart)** — the logo mark widget (rounded
  green-gradient square, white ear glyph, orange dot).
